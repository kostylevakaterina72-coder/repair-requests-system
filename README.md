## 📋 О проекте

Веб-приложение для автоматизации процесса учёта и обработки заявок на ремонт электрооборудования на предприятии. Позволяет создавать, отслеживать и анализировать заявки в реальном времени.

### ✨ Основные возможности

| Функция | Описание |
|---------|----------|
| ➕ **Создание заявок** | Добавление заявок с указанием оборудования, типа, описания и приоритета |
| 👁️ **Просмотр** | Табличное отображение всех заявок с детальной информацией |
| ✏️ **Редактирование** | Изменение параметров заявки, обновление статуса |
| 🗑️ **Удаление** | Удаление устаревших или ошибочных заявок |
| 🔍 **Поиск** | Быстрый поиск по наименованию оборудования |
| 🏷️ **Фильтрация** | Фильтрация заявок по статусу выполнения |
| 📊 **Статистика** | Визуальное отображение ключевых показателей |
